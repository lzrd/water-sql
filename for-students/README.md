## What You'll Learn

- **SQL**: Query large datasets (3+ million water quality measurements)
- **Python**: Data analysis with pandas, matplotlib, seaborn
- **Statistics**: Calculate averages, identify trends, spot outliers
- **Geographic Analysis**: Distance calculations, interactive maps
- **Data Science Skills**: Work with real-world data used by scientists and engineers

## The Database

You have access to EPA STORET water quality data with:
- **3+ million measurements** from thousands of monitoring stations
- **Parameters**: Temperature, dissolved oxygen, pH, turbidity, and more
- **Time span**: Decades of historical data
- **Coverage**: Entire state (Washington, Oregon, Illinois, etc.)

## Your Tools

- **SQLite**: A file-based database (no server needed!)
- **Python**: For analysis and visualization
- **Your curiosity**: The best tool of all

## Example Questions You Can Answer

- How has water temperature changed over the last 20 years?
- Which rivers have the best water quality?
- Are there seasonal patterns in dissolved oxygen levels?
- How does elevation affect water temperature?
- Where are the coldest/warmest monitoring stations?
